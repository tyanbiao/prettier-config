# @hyperbolajs/prettier-config

wwwatson's personal [prettier](https://prettier.io/) config

## Usage

**Install**

```shell
$ yarn add -D @hyperbolajs/prettier-config
```

**Edit `package.json`**

```jsonc
{
    // ...
    "prettier": "@hyperbolajs/prettier-config"
}
```
